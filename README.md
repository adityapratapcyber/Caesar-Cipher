# Caesar Cipher

This is a simple Python project that encrypts text using the Caesar Cipher technique.

I created this project to understand the basics of encryption and cryptography.

## Features

- Encrypts text using a fixed shift value
- Simple and easy-to-understand implementation
- Beginner-friendly project

## Example

Input:
Hello

Output:
Khoor

## Technologies Used

- Python

## What I Learned

- Basic cryptography concepts
- String handling in Python
- Loops and character manipulation
